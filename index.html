Project Overview

The Math Quiz Challenge is a web-based interactive quiz designed to enhance mathematical skills through dynamic problem-solving. Developed using HTML, CSS, and JavaScript, it generates randomized math problems with multiple operands and operators, ensuring a challenging and engaging experience. Users receive real-time validation and feedback, with intuitive navigation controls for flexibility.

Q/A asked by Ma'am and Group-7

1. Where are the user's answers stored?

The user's answers are stored in the userAnswers array. As users submit their responses, their input is pushed into this array. If a question is skipped, null is stored to indicate that no answer was provided. This allows the system to track and evaluate user performance during the result calculation.

2. How does the UI transition from question display to results?

The transition from the question display to the result screen is managed using DOM manipulation. When the quiz reaches the last question, the function showResults() is triggered. This function hides the quiz container (#quiz_container) and displays the result container (#result-container).

3. How are the mathematical questions generated with operators?

The questions are dynamically generated using the generateQuestions() function. It randomly selects three numbers between -10 and 10 and assigns two different mathematical operators (+, -, *, %) using the Math.random() function. The numbers are formatted properly to handle negative values, and the final question string is stored in the questions array.
